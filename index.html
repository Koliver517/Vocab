UNIT 8 VOCABULARY



Web Table
An HTML Structure that consisits of mulitple table rows and 1 or more columns, in which each row contains one or more table cells.
Rows
Vertical cells in a table
Columns
Horizontal cells in a table
Header cells
Contain content placed at a column top or begginign of a row, default text is bold and centered, is the th element
Data Cells
Contains content within columns or rows, default text is aligned to the left of a cell, is the td element
CSS border Property
Is used to add borders to any part of a web table, the two options are seperate or collapsed borders
Collapsed Borders Model
Borders from adjacent elements are merged to form a single border
Spanning Cells
Single cell that occupies more than one cell row and or column, created by adding rowspan/columnspan to td/th elements
Visual ViewPort
The styles able to be seen on a screen
Float Elements
Depending on the style values determines if and how much an element is positioned to the right or the left.
Flexboxes
A layout mode that allows elements to align and distribute space within a container
Data Attribute
A data attribute is a characteristic of data that sets it apart from other data, such as location, length, or type.
Block Elements
A block-level element is any element that starts a new line, like a paragraph, and uses the full width of the page or container.
Two Column Layout
A layout that is featured as two columns

UNIT 7 VOCABULARY



Flexible Layout
A layout type that automatically adjusts to screens of different widths
Responsive Images
Images that resize based on the size of the viewing device
Media Queries
Determine the properties of the device rendering the page so that appropriate designs can be delivered to specific devices
Not
A keyword that reverts the meaning of an entire media query
Only
A keyword that prevents older browsers that do not support media queries with media features from applying the specified styles, which has no effect on modern browsers
And
A keyword that combines a media feature with a media type or other media features
Mobile First
A principle in which the overall page design starts with base styles that apply to all devices followed by style rules specific to mobile devices
Layout viewport
Contains the entire content of the page, some of which may be hidden from the user
SDK(Software Development Kit)
An SDK is a collection of software used for developing applications for specific device or operating system
Main Axis
Can point in either the Horizontal or Vertical direction
Cross Axis
Is perpendicular to the main axis and is used to define the height or width of each item

Vendor-Prefix
A property that is used to implement new or proprietary CSS features, prior to final clarification/definition by the W3
-webkit
The vendor-prefix property for Chrome and Safari
-moz
The vendor-prefix property for Firefox
-ms
The vendor-prefix property for Internet Explorer
-o
The vendor-prefix property for Opera
3D Tansformation
Is a change that invloves three spatial axes, x-axis, y-axis, and the z-axes
X-axis
An axis that runs horizontally across the page
Y-axis
An axis that runs Vertically across the page
Z-axis
An axis that comes straight out of the page toward and away from the viewer
Perspective
A measure of how rapidly objects appear to recede from the viewer in a 3D space, which is used when only one object needs to be transformed in the 3D space
Vanishing Point
The perspective point that appears to converge at a point
Color Gradient
One color gradually blends into another color or fades away if transparent colors are used, and can be used to modify a background color
Linear Gradient
Background colors transition from a starting color to an ending color along a straight line, the default direction is vertical, starting from top and moving to bottom
Radial Gradient
It is a color gradient that starts from a central point, it proceeds outward in a series of concentric circles or ellipses
Border image
A border based on a graphic image
Watermarks
Translucent graphics displayed behind a content, whihc can be created using fixed background, and is often used to indicate that a content material is copyrighted
Tiling
The process of repeating an image both vertically and horizontally until the background is filled

Reset style sheet
A Style sheet that superseeds a browser's default styles and provides a consistent starting point for page design, the first style rule in a sheet is the display property used to display HTML5 structural elements
Fixed Layout
Size of the page and page elements are fixed, usually using pixels as the unit of measure
Fluid Layout
The width of the page elements are set as a percent of the available screen width
Elastic Layout
Images and text are always sized in proportion to each other in em units
Responsive Design
The layout and design of a page changes in response to the devices that is rendering it
Content Box Model
The width property refers to the width of an element content only, additonal space include padding or borders
Border Box Model
The width property is based on the sum of the content, padding, and border spaces, additional space taken up by padding and border is subtracted from space given to the content
Container Collapse
An empty container with no content, elements in the container are floated
Fr Unit
Represents the fraction of available space left on the grid after all other rows or columns have attained their maximum allowable size
Grid Cell
A space in a row or column that is open for content
Fixed Grids
Every column has a fixed position, widths of the columns and margins are specified in pixels
Fluid Grids
Provides more support across different devices with different screen sizes, column width is expressed in percentages
Framework
Is a software package that provides a library of tools to design a website, which includea style sheets for grid layouts and built-in scripts to provide support for a variety of browsers and devices
Bootstrap
Bootstrap is a free and open-source CSS framework directed at responsive, mobile-first front-end web development.
YAML4
YAML (Yet Another Multicolumn Layout) is a cross-browser CSS framework, whihc allows web designers to create a low-barrier website with comparatively little effort
960 Grid System
The 960 Grid System is an effort to streamline web development workflow by providing commonly used dimensions, based on a width of 960 pixels.
Foundation 3
Zurb Foundation 3 is a responsive front-end framework to build web sites and apps fast
Static Positioning
The element is places where it would have fallen naturally within the flow of the document
Relative Positioning
The element is moved out of its normal position in the document flow
Absolute Positioning
The element is placed at specific coordinates within containers/dd>
Fixed positioning
Fixes an object within a browser window to avoid its movement
Inherited Positioning
Allows an element to inherit the position value of its parent Elements
Clip
Defines a rectangular region through which an element's content can be viewed, anything that lies outside the boundary of the rectangle is hidden

CSS3
The latest version of the CSS language which is built upon several modules, each focused on a separate desgin topic
Modules
A pre-programmed package of software code that can be inserted into a webpage, and are compsoed of templates and/or functions for you to use when building your website
Browser Styles/User Agent Styles
Styles built into the browser
User-defined Styles
Styles defined by a user based on the configuration setting of the user's browser
External Styles
Styles created by a website author placed within a CSS file and linked to the page
Browser Extensions
An extended library of style properties in the browser
Vendor Prefix
Indicates the browser vendor that created and supports the style property
Style Inheritance
The process in which properties are passed from a parent element to its children
Hue
Tint of a color, represented by a direction on a color wheel
Saturation
Measures the intensity of a color and ranges from 0%(no color) up to 100%(full color)
Lightness
Measures the brightness of a color and ranges from 0%(black) up to 100%(white)
Opacity
Defines how solid a color appears
Contextual Selector
Specifies the context under which a particular page element is matched
Parent Element
A parent element is usually the containing element, with the elements inside being its child or children.
Child elements
An element that is an immediate descendent of another element or nested within another element is called a child.
Descendant elements
An descendant refers to any element that is connected but further up the document tree, no matter how many levels higher.
Wildcard Selector
A selector that matches any element using the * character
Sibling Selectors
Selects elements based on elements that are adjacent to them in the document hiearchy
ID
Identifies specific elements within the document, using the # character
Class
Identifies a group of elements that share a similar characteristic or property, using the . character
